# B2B Onboarding

Automated B2B customer onboarding workflow with KRS verification and risk assessment.

> **Self-Sufficient DSL**: All source code generated from `contracts/onboarding.reclapp`.

## Quick Start

```bash
reclapp generate contracts/onboarding.reclapp --output target/
cd target && docker-compose up -d
```

## Features

- Automated KRS/NIP verification
- Document collection workflow
- Risk scoring integration
- Multi-step approval process
