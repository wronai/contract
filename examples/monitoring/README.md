# Contractor Monitoring

Real-time contractor risk monitoring with financial data integration.

> **Self-Sufficient DSL**: All source code generated from `contracts/contractor-monitoring.reclapp`.

## Quick Start

```bash
reclapp generate contracts/contractor-monitoring.reclapp --output target/
cd target && docker-compose up -d
```

## Features

- Real-time financial monitoring
- Risk score calculation
- Automated alerts
- Integration with KRS and financial data providers
