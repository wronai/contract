# Analytics Reporting

Business analytics and reporting dashboard with scheduled reports.

> **Self-Sufficient DSL**: All source code generated from `contracts/analytics.reclapp`.

## Quick Start

```bash
reclapp generate contracts/analytics.reclapp --output target/
cd target && docker-compose up -d
```

## Features

- Custom report builder
- Scheduled report generation
- Export to PDF/CSV/Excel
- Dashboard widgets
